## Tools and Libraries
* Python
* Pandas
* NumPy
* yfinance
* Scikit-learn
* Matplotlib

Apple stock market dataset downloaded from Yahoo Finance.

## Features Used
* Open
* High

## Work Done

* Performed data preprocessing
* Trained Linear Regression model
* Predicted stock prices
* Compared actual and predicted prices using graphs

## Conclusion
The Linear Regression model successfully predicted stock prices with good accuracy. The predicted values were close to the actual prices.
